# Wikipedia Melhorada

Este é um projeto web criado para o desafio 3 do curso de Formação HTML Web Developer. A página é uma simulação da Wikipédia, com o tema "Sistema Solar", e inclui informações sobre os planetas.

## 📑 Estrutura do Projeto

O projeto está dividido em três partes principais:

1. **Cabeçalho (Header)**: Exibe o logo do site.
2. **Menu Lateral (Sidebar)**: Contém uma lista de links que representam diferentes partes do Sistema Solar, como o Sol, os planetas, e o Sistema Solar em si.
3. **Conteúdo Principal (Main Content)**: Apresenta informações sobre o Sistema Solar, sua formação e o processo de formação dos planetas.


## 🚀 Funcionalidades

- **Navegação por links**: O menu lateral permite ao usuário navegar por diferentes páginas do sistema solar.
- **Menu âncora**: Um menu interno na parte inferior que permite a navegação direta para seções específicas do artigo.
- **Imagens com legenda**: As imagens contam com legendas para descrever o conteúdo e melhorar a experiência do usuário.
  
## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura e organização da página.
- **CSS3**: Estilos visuais e layout.


## ⚠️ Nota

Este projeto é apenas uma simulação educacional e não faz parte do site oficial da Wikipédia.
